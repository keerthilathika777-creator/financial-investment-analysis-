# financial-investment-analysis-

## 📌 Project Overview
This project focuses on analyzing investment behavior and financial preferences using Power BI. The dashboard provides insights into investment choices, savings objectives, investment duration, monitoring frequency, reasons for investment, and sources of investment information.

The project was developed as part of a Power BI Internship task to demonstrate data analysis, visualization, and dashboard development skills.

---

## 🎯 Objectives
- Understand and summarize investment dataset statistics
- Analyze gender-based investment preferences
- Study savings objectives and investment avenues
- Examine investment duration and monitoring frequencies
- Analyze reasons behind investment decisions
- Identify common sources of investment information
- Build an interactive and professional Power BI dashboard

## 📊 Analyses Performed

### 1. Data Exploration & Summary
- Imported and explored dataset structure
- Verified data types and checked missing values
- Calculated summary statistics
- Created initial visualizations

### 2. Gender-Based Analysis
- Compared investment preferences between genders
- Visualized Equity, Mutual Funds, Bonds, and Gold investments

### 3. Objective Analysis
- Analyzed relationship between savings objectives and investment choices
- Compared preferred investments for different financial goals

### 4. Investment Duration & Frequency Analysis
- Visualized investment duration distribution
- Analyzed investment monitoring frequencies
- Examined impact on investment preferences

### 5. Reasons for Investment
- Identified common reasons for investing
- Analyzed trends such as Better Returns and Tax Benefits

### 6. Source of Information Analysis
- Examined where investors gather investment information
- Compared Internet, Newspapers, and other information sources

### 7. Final Interactive Dashboard
- Combined all analyses into a single dashboard
- Added slicers and interactive filtering
- Designed a professional dashboard layout

## 🔍 Key Insights
- Mutual Funds and Equity Markets were among the most preferred investment avenues.
- Male investors showed higher interest in Equity investments.
- Most respondents preferred medium-term investments (1–3 years).
- Better Returns was a major investment motivation.
- Internet was the primary source of investment information.
- Frequent investment monitoring was associated with Equity-based investments.
